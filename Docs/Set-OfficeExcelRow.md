# Set-OfficeExcelRow
## SYNOPSIS
Writes a row of values to the current worksheet.

## SYNTAX
### __AllParameterSets
```powershell
Set-OfficeExcelRow [-Row] <int> [[-Values] <Object[]>] [-StartColumn <int>] [-Height <double>] [-ClearHeight] [-AutoFit] [-Hidden <bool>] [-Bold <bool>] [-Italic <bool>] [-Underline <bool>] [-WrapText <bool>] [-FontName <string>] [-BackgroundColor <string>] [-FirstColumn <int>] [-LastColumn <int>] [<CommonParameters>]
```

## DESCRIPTION
Writes a row of values to the current worksheet.

## EXAMPLES

### EXAMPLE 1
```powershell
PS> ExcelSheet 'Data' { Set-OfficeExcelRow -Row 2 -Values 'North', 1200 }
```

Writes two values into row 2, columns A and B.
