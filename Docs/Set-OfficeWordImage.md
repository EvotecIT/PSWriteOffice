# Set-OfficeWordImage
## SYNOPSIS
Updates OfficeIMO Word image sizing, wrapping, crop, and metadata.

## SYNTAX
### __AllParameterSets
```powershell
Set-OfficeWordImage [-Image] <WordImage> [-Width <double>] [-Height <double>] [-Wrap <WrapTextImage>] [-CropTop <int>] [-CropBottom <int>] [-CropLeft <int>] [-CropRight <int>] [-Rotation <int>] [-Opacity <int>] [-HorizontalFlip <bool>] [-VerticalFlip <bool>] [-Title <string>] [-Description <string>] [-Hidden <bool>] [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Updates OfficeIMO Word image sizing, wrapping, crop, and metadata.

## EXAMPLES

### EXAMPLE 1
```powershell
PS> $doc = Get-OfficeWord -Path .\Report.docx
            $doc |
                Get-OfficeWordImage |
                Set-OfficeWordImage -Width 320 -Wrap Square -Title 'Report image' -Description 'Image used in the report'
            $doc | Save-OfficeWord -Path .\Report-Images.docx
```

Gets OfficeIMO image objects from an open document, applies thin image metadata and layout updates, and saves the document.
