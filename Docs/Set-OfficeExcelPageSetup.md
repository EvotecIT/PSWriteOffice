# Set-OfficeExcelPageSetup
## SYNOPSIS
Configures page setup options on a worksheet.

## SYNTAX
### Context (Default)
```powershell
Set-OfficeExcelPageSetup [-FitToWidth <uint>] [-FitToHeight <uint>] [-Scale <uint>] [-PassThru] [<CommonParameters>]
```

### Document
```powershell
Set-OfficeExcelPageSetup -Document <ExcelDocument> [-Sheet <string>] [-SheetIndex <int>] [-FitToWidth <uint>] [-FitToHeight <uint>] [-Scale <uint>] [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Configures page setup options on a worksheet.

## EXAMPLES

### EXAMPLE 1
```powershell
PS>ExcelSheet 'Data' { Set-OfficeExcelPageSetup -FitToWidth 1 -FitToHeight 0 }
```

Fits the sheet to one page wide and unlimited height.
