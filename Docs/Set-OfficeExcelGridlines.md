# Set-OfficeExcelGridlines
## SYNOPSIS
Shows or hides worksheet gridlines.

## SYNTAX
### Context (Default)
```powershell
Set-OfficeExcelGridlines [-Show] [-Hide] [-PassThru] [<CommonParameters>]
```

### Document
```powershell
Set-OfficeExcelGridlines -Document <ExcelDocument> [-Sheet <string>] [-SheetIndex <int>] [-Show] [-Hide] [-PassThru] [<CommonParameters>]
```

## DESCRIPTION
Shows or hides worksheet gridlines.

## EXAMPLES

### EXAMPLE 1
```powershell
PS>ExcelSheet 'Data' { Set-OfficeExcelGridlines -Hide }
```

Turns off gridlines for the sheet.
