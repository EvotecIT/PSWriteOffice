# Remove-OfficePdfAnnotation
## SYNOPSIS
Removes PDF annotations matching friendly filters.

## SYNTAX
### __AllParameterSets
```powershell
Remove-OfficePdfAnnotation [-Path] <string> [-OutputPath] <string> [-ObjectNumber <int>] [-PageNumber <int>] [-Subtype <string>] [-KeepPopups] [-PassThruReport] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Removes PDF annotations matching friendly filters.

## EXAMPLES

### EXAMPLE 1
```powershell
Remove-OfficePdfAnnotation -Path 'C:\Path'
```
