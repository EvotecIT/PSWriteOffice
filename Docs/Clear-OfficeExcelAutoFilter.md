# Clear-OfficeExcelAutoFilter
## SYNOPSIS
Clears any AutoFilter on the current worksheet.

## SYNTAX
### Context (Default)
```powershell
Clear-OfficeExcelAutoFilter [<CommonParameters>]
```

### Document
```powershell
Clear-OfficeExcelAutoFilter -Document <ExcelDocument> [-Sheet <string>] [-SheetIndex <int>] [<CommonParameters>]
```

## DESCRIPTION
Clears any AutoFilter on the current worksheet.

## EXAMPLES

### EXAMPLE 1
```powershell
PS>ExcelSheet 'Data' { Clear-OfficeExcelAutoFilter }
```

Removes filter dropdowns and criteria.
