---
title: "Install PSWriteOffice"
description: "Install PSWriteOffice from the PowerShell Gallery."
layout: docs
---

Install PSWriteOffice before trying the curated document automation examples.

```powershell
Install-Module -Name PSWriteOffice -Scope CurrentUser
Import-Module PSWriteOffice
```

PSWriteOffice does not require Microsoft Office to be installed for the core document generation workflows.
